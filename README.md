# Wireless-Communication-Project
Code for 'Bubbles' Area Detector implemented on ESP32-C6.
This system aims to inform the user about the whereabouts of ‘Bubbles’ with respect to a specific area. If ‘Bubbles’ is within the Bluetooth threshold range, it suggests that ‘Bubbles’ has entered the restricted area. A notification will be sent via rainmaker as a result. If ‘Bubbles’ exceeds the Bluetooth threshold range, it suggests that ‘Bubbles’ has left the restricted area.
